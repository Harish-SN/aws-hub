AWS Security Services

This list covers AWS security services used for threat detection, vulnerability management, data protection, network defense, identity & access control, compliance, and investigation.

Amazon GuardDuty

Type: Managed Threat Detection
Operation: Automatic

  • Detects active attacks and suspicious behavior
  • Analyzes CloudTrail, VPC Flow Logs, DNS logs
  • No agents required

Amazon Inspector

Type: Vulnerability Management
Operation: Automatic

  • CVE-based scanning for EC2, ECR, Lambda
  • Severity-based findings

Amazon Macie

Type: Sensitive Data Discovery
Operation: Automatic

  • Detects PII and sensitive data in S3
  • ML-powered classification

AWS Security Hub

Type: Security Posture Management
Operation: Automatic

  • Aggregates findings from GuardDuty, Inspector, Macie
  • Maps to CIS, PCI DSS, and other standards

AWS Config

Type: Configuration Compliance
Operation: Automatic (rule-based)

  • Tracks resource configuration changes
  • Detects compliance drift

Amazon Detective

Type: Security Investigation
Operation: Manual (analyst-driven)

  • Investigates GuardDuty findings
  • Graph-based visualization

AWS CloudTrail

Type: API Auditing & Governance
Operation: Automatic

  • Records all AWS API calls
  • Foundation for security forensics

AWS IAM & Access Analyzer

Type: Identity & Access Security
Operation: Policy-driven

  • Controls authentication & authorization
  • Detects public or cross-account access

AWS WAF

Type: Web Application Firewall
Operation: Policy-driven

  • Blocks SQLi, XSS, bots
  • Works with CloudFront, ALB, API Gateway

AWS Shield

Type: DDoS Protection
Operation: Automatic

  • Shield Standard (free, always on)
  • Shield Advanced (enhanced protection)

AWS Network Firewall

Type: Stateful VPC Firewall
Operation: Policy-driven

  • Deployed inside VPC subnets
  • Deep packet inspection

AWS Firewall Manager

Type: Central Policy Management
Operation: Policy-driven

  • Org-wide WAF, Shield, SG policies
  • Requires AWS Organizations

AWS KMS & Certificate Services

Operation: Automatic

  • KMS: Encryption key management
  • ACM: Managed TLS certificates
  • Private CA: Managed PKI

AWS Secrets Manager & Parameter Store

Operation: Automatic

  • Secure storage of secrets
  • Automatic rotation support
