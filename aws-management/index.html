AWS Management & Governance

Covers AWS management, operations, governance, automation, and account control services used for enterprise-scale environments, multi-account strategies, and operational excellence.

AWS Organizations

Type: Multi-Account Management

  • Service Control Policies (SCPs).
  • Organizational Units (OU).
  • Consolidated billing.

AWS Control Tower

Type: Landing Zone & Governance Automation

  • Automated multi-account setup.
  • Preventive & detective guardrails.
  • Built on Organizations.

AWS Systems Manager (SSM)

Type: Operations & Fleet Management

  • Run Command & Automation.
  • Patch Manager.
  • Session Manager (no SSH).
  • Parameter Store.

AWS AppConfig

Type: Runtime Configuration Management

  • Feature flags & dynamic configuration.
  • Safe rollouts with validators.
  • Part of Systems Manager.

AWS CloudFormation

Type: Infrastructure as Code (IaC)

  • Declarative YAML/JSON templates.
  • Drift detection.
  • StackSets for multi-account deployments.

AWS CDK (Cloud Development Kit)

Type: Code-Based IaC

  • TypeScript, Python, Java, Go.
  • Generates CloudFormation.
  • Reusable constructs.

AWS Service Catalog

Type: Governed Self-Service Provisioning

  • Approved infrastructure products.
  • Controlled deployments for teams.

AWS Resource Explorer

Type: Global Resource Discovery

  • Search resources across regions/accounts.
  • Tag-based discovery.

AWS License Manager

Type: License Governance

  • Track Windows, SQL, BYOL licenses.
  • Integrates with Systems Manager.

AWS Service Quotas

Type: Limits & Quota Management

  • View service limits.
  • Request quota increases.

AWS Trusted Advisor

Type: Optimization & Governance

  • Cost, performance, security recommendations.
  • Support-plan dependent.

AWS OpsWorks (Legacy)

Type: Configuration Management

  • Chef & Puppet automation.
  • Avoid for new workloads.

AWS Well-Architected Tool

Type: Architecture Review

  • 6-pillar framework.
  • Workload reviews & improvements.

AWS Health Dashboard

Type: Service Health & Alerts

  • Personalized outage notifications.
  • EventBridge integration.
