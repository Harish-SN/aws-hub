AWS Monitoring, Auditing & Cost Governance

This 2025 edition covers AWS monitoring, observability, auditing, cost management, and governance services to ensure reliability, cost control, and visibility across your cloud environment.

Amazon CloudWatch

Type: Monitoring & Observability
Use Cases: Metrics, logs, alarms, dashboards, traces

Overview

CloudWatch collects logs and metrics from AWS services and applications.

  • CloudWatch Metrics, Logs, Alarms, Dashboards.
  • Log Insights for log queries.
  • X-Ray integration for tracing.

CloudWatch Synthetics

Type: Synthetic Testing (Canaries)
Use Cases: API health checks, uptime monitoring

Overview

Runs automated canary scripts to monitor websites/APIs 24√ó7.

  • Detect UI failures.
  • Latency monitoring.
  • Integrates with CloudWatch alarms.

CloudWatch RUM

Type: Real User Monitoring
Use Cases: Track frontend performance & user behavior

Overview

Collects performance data from real user browsers/mobile apps.

  • Frontend errors, page loads, JS issues.
  • Correlation with backend traces.

CloudWatch Application Signals

Type: Service-Level Observability
Use Cases: SLO/SLI monitoring for microservices

Overview

Provides golden metrics: availability, latency, errors, throughput.

  • Auto-detects service dependencies.
  • Integrated with X-Ray and Lambda.

Amazon EventBridge

Type: Event Bus / Event Routing
Use Cases: Automation, notifications, serverless workflows

Overview

EventBridge routes events between AWS services and your applications.

  • Automates CloudWatch / CloudTrail events.
  • Integrates with SaaS tools.
  • Supports scheduled cron rules.

AWS CloudTrail

Type: Audit & API Activity Logging
Use Cases: Forensics, security investigations

Overview

CloudTrail logs every API call made through the console, CLI, SDK.

  • CloudTrail Lake for SQL-style queries.
  • Logs to S3 for long-term retention.
  • Integrated with GuardDuty + Config.

AWS Config

Type: Resource Compliance & Drift
Use Cases: Governance, audits, resource history

Overview

Tracks the configuration history of AWS resources and checks for compliance.

  • Detects non-compliant resources.
  • Records configuration change timeline.
  • Integrates with Organizations.

AWS Cost Explorer

Type: Cost Analytics
Use Cases: Cost breakdowns, forecasting

Overview

Provides cost visibility across accounts, services, and usage.

  • Daily/Monthly spend tracking.
  • RI & Savings Plan recommendations.

AWS Budgets

Type: Cost Control
Use Cases: Alerts for cost thresholds

Overview

Set custom cost/usage budgets with alerts.

  • Email/SNS alerts.
  • Supports RI & Savings Plan tracking.

AWS Trusted Advisor

Type: Optimization Recommendations
Use Cases: Performance, cost, security

Overview

Advisor gives real-time best-practice recommendations.

  • Cost optimization.
  • Security checks like public S3.
  • Performance boosts.

AWS Health Dashboard

Type: Account & Service Health
Use Cases: Outage alerts, planned maintenance

Overview

Shows personalized AWS outage and service impact notifications.

  • Organization-wide health visibility.
  • Integration with EventBridge.
