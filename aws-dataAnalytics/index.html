AWS Data Analytics

AWS provides multiple purpose-built analytics services for streaming, querying, processing, search, observability, and visualization.

Amazon Kinesis

Type: Real-time Data Streaming
Serverless: âœ… Yes
Scaling: Auto or provisioned capacity per stream
Use Cases: IoT streams, real-time analytics, logs, clickstreams

Overview

Kinesis enables ingestion, buffering, processing, and routing of real-time streaming data.

  • Sub-second latency streaming.
  • Integrates with Lambda, Firehose, OpenSearch, Redshift.
  • Modules: Data Streams, Firehose, Data Analytics (Flink), Video Streams.

Amazon Athena

Type: Serverless Query Engine (SQL on S3 & Lakehouse)
Serverless: âœ… Yes
Scaling: Auto-scales per query
Use Cases: Ad-hoc SQL, querying Iceberg tables, data lake analytics

Overview

Athena is serverless Presto/Trino that queries data directly from S3 or Iceberg.

  • Zero infrastructure to manage.
  • Supports Iceberg, Parquet, JSON, CSV, ORC, Avro.
  • Uses Glue Data Catalog for metadata.

Amazon Redshift

Type: Cloud Data Warehouse
Serverless: âœ… Yes (Redshift Serverless)
Scaling: Auto-scaling or cluster node-based
Use Cases: Enterprise BI, high-concurrency SQL

Overview

Redshift is a massively parallel, columnar data warehouse.

  • Redshift Serverless scales automatically.
  • Redshift Spectrum queries S3/Iceberg without loading.
  • Integrates with S3, Glue, QuickSight, Firehose.

AWS Lake Formation

Type: Data Lake Governance & Access Control
Serverless: âœ… Yes
Use Cases: Lakehouse security, multi-team access, fine-grained permissions

Overview

Lake Formation secures data lakes and lakehouse tables (Iceberg) and centralizes permissions.

  • Provides table-, column-, and row-level security.
  • Works with Athena, Redshift Spectrum, EMR, Glue.

AWS Glue

Type: ETL & Data Integration
Serverless: âœ… Yes
Use Cases: ETL pipelines, data catalog, data prep

Overview

Glue prepares data for analytics using Spark-based jobs and schema discovery.

  • Glue Studio for visual ETL.
  • Glue DataBrew for no-code data cleaning.
  • Glue Data Catalog is used across Athena/Redshift/EMR.

Amazon QuickSight

Type: Serverless BI Dashboards
Use Cases: Business dashboards, ML insights, embedded analytics

Overview

QuickSight provides interactive dashboards with SPICE in-memory caching.

  • Integrates with Redshift, Athena, RDS, S3.
  • ML-powered anomaly detection.
  • Pay-per-session pricing.

Amazon EMR

Type: Big Data Processing (Spark, Hadoop, Hive)
Serverless: âœ… Yes (EMR Serverless)
Use Cases: ETL at scale, machine learning, big data pipelines

Overview

EMR is the most cost-efficient service for Spark/Hadoop workloads.

  • Supports Spark, Presto, Hive, Hadoop, Trino.
  • EMR Serverless removes cluster management.
  • Deep integration with Glue & Lake Formation.

Amazon OpenSearch Service

Type: Search, Log Analytics, Vector Search
Serverless: âœ… Yes
Use Cases: Full-text search, log analytics, observability dashboards

Overview

OpenSearch powers search workloads and log analytics.

  • Managed clusters or serverless collections.
  • Supports Kinesis, Firehose, CloudWatch Logs ingestion.
  • Supports vector search (kNN) for ML workloads.

Amazon MSK (Managed Kafka)

Type: Streaming Platform (Kafka)
Serverless: âœ… MSK Serverless
Use Cases: High-throughput streaming, Kafka-based apps

Overview

A fully managed Apache Kafka service.

  • Fully manages Kafka brokers and Zookeeper.
  • Can run in VPC, multi-AZ, secure networking.
  • MSK Serverless scales compute & storage automatically.
  • Compatible with all Kafka producers/consumers.

Amazon Managed Service for Apache Flink

Type: Real-time Stream Processing (Apache Flink)
Serverless: âœ… Yes
Use Cases: Stream transformations, real-time enrichment, anomaly detection

Overview

Runs any Apache Flink application fully managed by AWS.

  • Write Flink apps in Java, Scala, Python, SQL.
  • Connects to Kinesis Data Streams & MSK.
  • Automatic scaling, checkpoints & state management.
  • Used for complex stream processing pipelines.
