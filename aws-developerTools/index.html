AWS CodeArtifact

AWS CLI

Type: Command Line Interface
Use Cases: Automation, scripting, CI/CD integration

Unified CLI for managing AWS services from terminal scripts and pipelines.

AWS SDKs

Type: Application SDK Libraries
Languages: Python, JavaScript, Java, Go, C#, Ruby

Enable applications to interact with AWS services programmatically.

AWS SAM (Serverless Application Model)

Type: Serverless Infrastructure as Code
Use Cases: Lambda, API Gateway, event-driven apps

Simplified IaC framework built on CloudFormation for serverless workloads.

AWS CDK (Cloud Development Kit)

Type: Infrastructure as Code
Languages: TypeScript, Python, Java, Go, C#

Define AWS infrastructure using real programming languages and generate CloudFormation templates.

AWS Systems Manager

Type: Operations & Automation
Use Cases: Patching, runbooks, Parameter Store

Central service for managing, automating, and operating AWS resources.

AWS X-Ray

Type: Distributed Tracing
Use Cases: Debugging microservices and latency issues

Traces requests across distributed systems and visualizes service maps.

AWS Fault Injection Service

Type: Chaos Engineering
Use Cases: Resilience and failure testing

Injects controlled faults to validate system reliability and recovery.
