üß© OSI Model (7 Layers)

Defines how data travels through networks ‚Äî from physical signals to application-level communication.

  1. 7. Application ‚Äì User interaction level. Protocols: HTTP, HTTPS, DNS, FTP, SMTP Role: Interfaces apps with the network (e.g., web browsing, email).
  2. 6. Presentation ‚Äì Data format & encryption. Protocols: SSL/TLS, JSON, XML Role: Converts, compresses, and encrypts data for secure transmission.
  3. 5. Session ‚Äì Connection management. Protocols: NetBIOS, RPC Role: Starts, maintains, and terminates sessions between hosts.
  4. 4. Transport ‚Äì Data reliability & flow control. Protocols: TCP, UDP Role: Ensures ordered delivery (TCP) or fast, connectionless transfer (UDP).
  5. 3. Network ‚Äì Logical routing & addressing. Protocols: IP, ICMP, BGP Role: Moves packets between networks based on IP addresses. Note: BGP ASN (Autonomous System Number) identifies routing domains. ‚Üí AWS ASN: 7224 | Customer ASN: 65000
  6. 2. Data Link ‚Äì Frame delivery between nodes. Protocols: Ethernet, ARP Role: Provides error detection, defines MAC addresses.
  7. 1. Physical ‚Äì Transmission medium. Components: Cables, Fiber, Radio waves Role: Converts bits into physical signals.
