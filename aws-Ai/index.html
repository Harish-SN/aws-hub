Amazon Bedrock

Type: Foundation Model Platform (GenAI)
Use Cases: Chatbots, summarization, RAG, agent automation

Overview

Bedrock provides access to foundation models from AWS, Anthropic, Meta, Mistral, Cohere, Stability, and more.

  • Model inference API for text, image, embeddings.
  • Bedrock Agents for automation workflows.
  • Knowledge Bases with vector search.

Amazon SageMaker

Type: Full ML Development Platform
Use Cases: Train, tune, deploy, MLOps pipelines

Overview

SageMaker is the most powerful managed ML platform on AWS.

  • Notebook Instances & Studio IDE.
  • Training jobs with distributed training.
  • Real-time, Batch, and Serverless endpoints.
  • Automatic Model Tuning (HPO).

Key MLOps Components

  • Feature Store: Centralized features for ML apps.
  • Pipelines: CI/CD automation for ML.
  • Clarify: Bias & explainability.
  • Canvas: No-code ML tool.
  • Data Wrangler: ML data prep tool.

Amazon Comprehend

Type: NLP Service
Use Cases: Sentiment, entity extraction, PII, topic modeling

Overview

Comprehend uses NLP to analyze and extract insights from text.

  • Sentiment analysis.
  • Entity recognition (names, places, dates).
  • PII detection.
  • Custom classifiers.

Amazon Rekognition

Type: Computer Vision
Use Cases: Face detection, object detection, moderation

Overview

Rekognition provides image & video analysis APIs.

  • Face recognition & comparison.
  • Label/object detection.
  • Content moderation.
  • Celebrity recognition.

Amazon Textract

Type: Document AI (OCR)
Use Cases: PDF extraction, invoices, forms

Overview

Textract extracts structured data from scanned documents.

  • OCR text extraction.
  • Form fields (key-value pairs).
  • Tables from documents.
  • Analyzes invoices and receipts.

Amazon Transcribe

Type: Speech-to-Text
Use Cases: Call centers, subtitles, audio analysis

Overview

Transcribe converts speech into accurate text.

  • Real-time and batch transcription.
  • Custom vocabulary.
  • Speaker identification.

Amazon Polly

Type: Text-to-Speech
Use Cases: Voice apps, narrators, alerts

Overview

Polly converts text into human-like speech.

  • Neural TTS voices.
  • Speech marks for animations.
  • Supports Lexicons & SSML for pronunciation and speech styling.

Amazon Translate

Type: Neural Machine Translation
Use Cases: Localization, multi-language apps

Overview

Translate converts text across many supported languages using neural networks.

  • Real-time & batch translation.
  • Custom terminology support to preserve domain-specific terms.
  • Can be integrated with API Gateway, Lambda, S3 for translation pipelines.

Amazon Lex

Type: Conversational AI
Use Cases: Chatbots, virtual assistants

Overview

Lex builds conversational interfaces using ASR + NLU.

  • Powered by same tech as Alexa.
  • Multi-turn conversations.
  • Integrates with Lambda, Polly and Connect for IVR/chatbots.

Amazon Kendra

Type: Enterprise Search (AI Search)
Use Cases: Knowledge base search

Overview

Kendra provides intelligent search across enterprise documents.

  • Semantic search.
  • Connectors for S3, SharePoint, Google Drive.

Amazon Connect

Type: Cloud Contact Center (CCaaS)
Use Cases: Customer support, IVR, omnichannel contact flows

Overview

Amazon Connect is a managed cloud contact center service that powers voice and chat customer interactions and integrates tightly with AWS AI/ML services.

  • Out-of-the-box contact flows, IVR, routing, and real-time metrics.
  • Integrates with Amazon Lex for conversational IVR and with Polly for TTS prompts.
  • Uses Amazon Transcribe and Contact Lens for real-time transcription, sentiment analysis, and post-call analytics.
  • Can stream call audio to Kinesis/S3 for analytics and to SageMaker or Comprehend for custom ML models.
  • Use cases: call center automation, chatbots, real-time agent assist, post-call quality analytics.

Amazon Personalize

Type: Recommendation Engine
Use Cases: Product, content, ranking recommendations

Overview

Personalize powers recommendation systems without ML expertise.

  • Real-time recommendations.
  • User personalization.
  • Behavioral ranking.

Amazon Forecast

Type: Time-Series Forecasting
Use Cases: Sales, demand, revenue forecasting

Overview

Forecast uses ML to predict future trends based on historical data.

  • Demand forecasting.
  • Supply chain planning.

Amazon Fraud Detector

Type: Fraud Detection ML
Use Cases: Transaction fraud, account takeovers

Overview

Detects fraudulent activities using ML models.

  • Real-time predictions.
  • Low-code rule engine.
